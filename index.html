4.What are the differences between var, let and const?

var

Keyword used to declare the variable

we are using let and const most of the time

  • we can declare the variable in one line and intialize it in another lin or in the same line
  • Two variables with the same name are possible with let
  • We can reassign/update the values of the variable which have already been initialized.
  • var is global scoped
  • var support hoisting

let

  • we can declare the variable in one line and intialize in other line or in the same line
  • Two variables with the same name are not possible with let
  • We can reassign/update the values of the variable which have already been initialized.
  • let is block scoped
  • does not support hoisting

const

  • We have to declare the variable and intialize it in the same line itself
  • Two variable with same name are not possible
  • We can not reassign/update the values of the variable which have already been initialized.
  • const is block scoped
  • does not support hoisting

undefined - when we try to access a variable that is declared but not initialized for var keywords.

null - Its a value that we initialize to a variable which will later be replaced with some other/true value

not defined - when we try to access a variable that is not declared.
